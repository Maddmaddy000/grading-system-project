What we do?

We are offering a automated text summarizer and question generating system that prepares questions on the basis of requirement as per the user. We offer different varieties of fill in the blanks type questions and a single type of W/H type question genartion in this system. We also offer a evaluation system that conduct the evaluation of these questions on user input and returns the user with the score.
